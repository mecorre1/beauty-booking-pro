## Story
As a hairdresser, I want a weekly calendar view where I can see and manage my schedule, exceptions and bookings in one place so that I can prepare ahead and stay in control of my time.

## Requirements
- Weekly calendar view in the back office
- Displays three layers: working hours, availability exceptions, bookings
- Working hours editable via DnD (drag to resize open/close times per day)
- Exceptions can be added by selecting a time range and marking it as closed (with optional comment)
- Bookings displayed as read-only blocks, clickable to view details
- Navigation between weeks (previous / next)

## Acceptance Criteria
- [ ] `GET /api/admin/schedule?week=YYYY-WW` returns weekly schedule, exceptions and bookings for the requested week
- [ ] `PUT /api/admin/schedule` updates open/close times for a given day
- [ ] `POST /api/admin/exceptions` creates an availability exception
- [ ] `DELETE /api/admin/exceptions/{id}` removes an exception
- [ ] Calendar renders all three layers visually distinct
- [ ] DnD resize updates open/close times and persists via API
- [ ] Clicking a booking block opens a detail panel
- [ ] Test: `test_get_schedule_returns_all_layers`
- [ ] Test: `test_create_exception_persists`
- [ ] Test: `AdminCalendarView` renders working hours, exceptions and bookings
