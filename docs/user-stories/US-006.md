## Story
As a client, I want to cancel my booking with my link so that the time becomes free for others.

## Requirements
- **Public:** `POST /api/public/bookings/by-token/{token}/cancel` (or `DELETE` with clear semantics).
- Set `status=cancelled`, restore slot `is_available=true`, clear buffer holds for that booking.

## Acceptance Criteria
- [x] Pytest: cancel restores availability; idempotent or second call returns clear error — document choice. *(Second cancel → 409 “already cancelled”.)*
- [x] Vitest: cancel confirmation UI. (`ClientEditView.spec.ts` — cancel button + `confirm`)

## Out of scope
- Admin-initiated cancel (can be added later as thin reuse).
