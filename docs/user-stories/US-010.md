## Story
As the hairdresser, I want to define weekly templates and apply them to a week so that I can generate many slots quickly.

## Requirements
- **SPEC “Weekly templates”:** ORM: `WeeklyTemplate`, `TemplateSlot` (day_of_week, start_time, duration).
- **Admin APIs:** create/update/list/delete templates; **apply** to a calendar week → **insert** `Slot` rows snapshot (template changes do not alter past applied weeks).
- Manual edit/delete of generated slots in a week can be **minimal** in this story (e.g. admin DELETE slot only) if full inline editor is too large — but SPEC mentions manual edit; include at least delete + “regenerate not supported” note, or small PATCH for slot times.

## Acceptance Criteria
- [x] Pytest: apply creates expected slot count; re-apply policy documented (reject vs replace). *(Reject apply if any booking exists in the week; else delete slots in week and insert.)*
- [x] Vitest: template form + apply week action. (`AdminScheduleView.spec.ts`)

## Out of scope
- Conflict resolution with existing bookings (document behavior: reject apply if conflicts).
